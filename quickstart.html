Quickstart

Self-host Metroflow in minutes.

One command brings up the workspace, crawler, and agent runtime. Point it at a warehouse and it starts building the graph.

Docker Compose

  • Works with free-tier Snowflake or local Postgres
  • Helm chart available for Kubernetes production deployments
  • Full control — nothing leaves your network
shell
$git clone https://github.com/metroflow/metroflow.git
$cd metroflow
$docker compose up
# workspace live at localhost:4200
