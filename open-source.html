Open Source

Built in public. Shipped for developers first.

Metroflow is open source under the Apache 2.0 license. Self-host it. Contribute to it. Run it in your environment with full control.

Why open source matters here

Data infrastructure is too important to be locked behind proprietary walls. We believe the best intelligence layer for enterprise data will be built in the open, with the people who actually run these systems every day.
